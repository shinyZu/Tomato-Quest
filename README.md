# TomatoQuest Game

<br>

### Assignment done under module Comparative Integrated Systems (CIS) in the final year at University of Bedfordshire.

<br>

## 4 themes implemented

<br>

1. Virtual Identity
2. Interoperability
3. Event-Driven Programming
4. Software Deisgn Principles
    - High Cohesion 
    - Low Coupling


<br>

## Technology Stack :

- Spring
- Maven
- MySQL
- React JS
- Axios
- MUI Components

<br>

## External APIs

- Tomato API
